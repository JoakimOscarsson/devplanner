# contracts-graph

Purpose:
- Define contracts for brainstorm canvases, skill graphs, canonical skills, and skill references.

Owns:
- Canvas, graph node, graph edge, skill, and duplicate-resolution types.
- Graph command/query envelopes and event names.
- Graph-domain error codes and duplicate-resolution payloads.

Does not own:
- Goal/planner state, progress projections, recommendation-provider execution, or MCP transport details.
